2. 의식은 있지만 움직이기 힘들고 패닉상태에 있음
	-> STT를 통해 값을 읽고 센서데이터를 통해 받은 정보를 토대로 상황에 맞는 안전 수칙 및 위로의 말 전달
	-> 이 후 TTS를 통해 생존자에게 전달함.
	-> "걱정 마세요. 구조 요청을 보냈습니다. 현재 CO2 수치가 높으니 자세를 낮추고 옷으로 코와 입을 막으세요."
3. 생존자의 위험 수치 및 상태 전달
	생존자의 상태에 따른 분류
	PIR센서를 통해 사람인지 ->
	TTS를 통해 사람 의식 확인 ->
	상태 :
		- Red (구조가 시급함)
			1. 말이 없는 상태 -> 의식불명 / 기절
			2. 대답은 할 수 있지만 움직일 수 없는 상태(패닉 / 상처) -> "다리를 다쳤어요 / 살려주세요"
			--> 현재 상태에서 해야할 안전 수칙 및 안심이 되는 말 TTS를 통해 전달하기.
		- Yellow ( 자체적 이동 가능 )
            -> 라이다 센서를 통해 그려놓은 map정보를 가져와 자율주행을 통해 출구로 사용자를 안내
            -> 로봇은 들어오며 센서 데이터(Co2 및 화염지역)을 기록했기 때문에 그 정보를 토대로 안전한 길로 안내
			-> 현재 센서에 맞는 안전 수칙에 대해 안내하며 유도
		- Green (사람없음)
			- 계속 사람을 찾음.
